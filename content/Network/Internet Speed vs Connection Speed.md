# Internet Speed vs Connection Speed

## Internet Speed
- Buy from ISP 
	- Download: 500Mb
	- Upload: 200Mb
	- 500 Baht/Month
- Shared (may slow at evening)
- Access Type (Stable Connection)
- Fixed-IP/Dynamic-IP
- Less Share, Premium

## Connction Speed
- Depend on both-end technology 
- WiFi (g) <-> WiFi (g, ac) = g (50Mb)
- Lan 100Mb <-> Lan 1Gb = Lan 100Mb
- WiFi 2.4Ghz vs 5Ghz
- Distance
