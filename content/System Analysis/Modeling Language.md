# Modeling Language

Focus on UML
- UML (Unified Modeling Language) is a standardized modeling language to describe the [[Object-Oriented]] type
- A modeling language is a formal language used to create models of systems, processes, or concepts. It provides a standardized way to represent and communicate complex information visually or textually. Modeling languages are used in various fields such as software engineering, systems engineering, business analysis, and more.
