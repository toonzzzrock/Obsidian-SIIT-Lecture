# System Modeling

- System modeling is the process of creating abstract representations of a system to understand, analyze, and design it effectively.
- It helps in visualizing the structure, behavior, and interactions of a system, making it easier to communicate ideas and identify potential issues.

## Information System Modeling Types
To model an information system, we can use different types of models based on the focus and perspective:
- Function-Driven/Procedure-Centered
	- Data-Flow Diagram
- Data-Driven
	- Entity-Relationship Diagram
- [[Object-Oriented]]
	- Static Model
		- [[Class Diagram]]
	- Dynamic Model
		- State Diagram
		- [[Sequence Diagram]]
